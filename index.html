How to Apply CSS to HTML

To include an external CSS file in an HTML document, you can use the <link> element in the <head> section:

<link rel="stylesheet" href="styles.css">

How to Use JavaScript in a Webpage

You can link to an external JavaScript file using the <script> tag:

<script src="script.js"></script>
